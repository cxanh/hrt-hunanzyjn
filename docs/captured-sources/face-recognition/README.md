# Face Recognition Source Notes

## 中文

该目录预留给与人脸识别检测相关的来源材料说明。

当前约定：

- 对应来源文件名：`common.js`
- 当前仅保存目录和说明文件
- 若后续需要补充内部记录，建议新增：
  - `source-note.md`：记录来源页面、抓取时间、入口位置、用途
  - `review-log.md`：记录审查结论、依赖项、已知限制

建议补充的信息：

- 来源页面 URL
- 抓取日期与操作者
- 入口 DOM 或触发流程
- 相关依赖库
- 观察到的功能模块概览

## English

This directory is reserved for notes related to captured source material for face-recognition checks.

Current convention:

- related source filename: `common.js`
- only the directory and explanatory notes are stored for now
- if more internal records are needed later, add:
  - `source-note.md` for source page, capture time, entry point, and purpose
  - `review-log.md` for review findings, dependencies, and known limitations

Suggested metadata:

- source page URL
- capture date and operator
- entry DOM node or trigger flow
- related dependency libraries
- high-level observed module summary
