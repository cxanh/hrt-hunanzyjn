# Captured Sources

## 中文

本目录用于存放从目标页面或相关前端资源中获取到的来源材料说明、采集记录和索引信息。

建议做法：

- 为每个来源文件建立一个单独子目录
- 记录来源页面、抓取时间、文件名和用途
- 如材料涉及身份校验、活体检测或其他敏感验证逻辑，优先只保存说明和元数据
- 如需保存原始内容，请先确认内部授权范围和最小化留存原则

## English

This directory is used for source-capture notes, intake records, and index information for materials obtained from target pages or related frontend assets.

Recommended practice:

- create a dedicated subdirectory for each captured file
- record source page, capture time, filename, and purpose
- for materials involving identity verification, liveness checks, or other sensitive validation logic, prefer storing notes and metadata only
- confirm internal authorization scope and data-minimization requirements before retaining any raw content
