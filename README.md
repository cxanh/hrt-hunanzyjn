- 仅限授权场景下的内部测试、学习与研究
- 不面向公开分发，不提供生产用途承诺
- 不应用于规避平台规则、身份校验或其他未授权行为

- `ChinahrtAutoplay-v2.js`

## 使用说明

本仓库当前主要用于保存脚本版本和研究记录。建议在受控测试环境中使用，并结合提交记录说明每次调整的背景、目标和验证结果。

## 开发建议

- 为每次脚本调整补充简要变更说明
- 在提交前记录适用页面、触发条件和已知限制
- 尽量保持脚本分版本存放，避免直接覆盖历史实现
