## 🌿 仓库定位

这是一个私有的“研究花园”，用于汇集关于 `chinahrt.com` 的内部测试脚本、灵感记录与迭代文档，旨在通过版本管理实现高效的团队协作与问题追踪。

---

## 使用方法

edge浏览器扩展->点击开发者模式->获取更多扩展->下载暴力猴/篡改猴->插件中点击添加脚本->添加ChinahrtAutoplay-v1.js

## ✨ 核心特性

- 💡 **多版本迭代**：提供 `v1` 与 `v2` 多个版本的脚本支持，便于进行对比测试与性能评估。
- 📖 **文档溯源**：独立的 `docs` 目录详细记录了研究结论及其对应的原始文件来源。
- 🎨 **标准模版**：使用 `templates` 快速初始化新脚本，确保项目结构的规范性。
- 🤝 **规范协作**：引入 `.github` 协作模版，统一 Issue 与 Pull Request 的处理流程。

## 🌱 维护规范

为保持项目结构的清晰与可维护性，请遵循以下约定：

1. **记录变更动机**：在调整脚本前，请记录变更的目的与预期的影响范围。
2. **保持提交原子性**：建议每次 Commit 仅包含一类逻辑变更。
3. **同步更新文档**：研究报告、已知限制及配置说明需及时同步至 `docs/` 目录。
4. **优先复用模版**：新增脚本组件时，请参考 `templates/` 中的预设规范。
